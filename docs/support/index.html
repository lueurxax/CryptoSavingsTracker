CryptoSavingsTracker Support

Need help with CryptoSavingsTracker?

CryptoSavingsTracker helps you track savings goals, manage assets, and follow real progress. If something is not working as expected, use the support options below.

Contact Support Project Page

Frequently Asked Questions

  • How do I add a goal?
    Open the Goals tab and use the add button in the top-right corner.
  • How do I track progress?
    Add assets to a goal, then review the goal detail and dashboard surfaces.
  • Can I use the app without an account?
    Yes. You can create goals and track progress locally without signing in.
  • How do I report a bug?
    Use the Contact Support button above and include screenshots, device model, and iOS version if possible.

Before You Contact Support

  • Make sure you are using the latest version available in TestFlight or the App Store.
  • Try restarting the app and reopening the affected screen.
  • Include the exact steps that caused the issue.
  • Attach a screenshot if the problem is visual or layout-related.
