# Navigation Release Evidence Templates

Use this folder as a starter for each new navigation migration wave package.

Recommended package layout:

- `guardrails-metrics-report.json`
- `go-no-go.md`
- `mod02-diff-report.json`
- `rollback-drill.md`
- `parity-matrix-report.json`
- `policy-report.json`
- `operational-hold-status.md` (required while Section 11 two-release guardrail criterion is pending)
- `hard-cutover-report.json`

After approval, copy final artifacts to `docs/release/navigation/latest/`.
